# 个人计算机清理技能

## 功能描述

该技能用于个人计算机清理，包含2个主要功能：
1. 磁盘优化 - 查找并展示/root路径下最大的3个文件，询问用户是否要清理；并包含特定openclaw文件的分析
2. 内存优化 - 查找并展示占用内存最大的3个进程，询问用户是否要清理

执行示例：
```bash
python scripts/cleaner.py disk  # 执行磁盘优化，包含openclaw文件分析结果
python scripts/cleaner.py memory  # 执行内存优化
```

## 安装依赖

无特殊依赖，使用Python标准库。

## 注意事项
1. 用户要求执行此技能的时候，直接执行磁盘优化和内存优化功能，无需用户确认，并打印执行过程中的全部print日志给用户，不要遗漏!包括openclaw文件搜索和分析的结果
2. 但是涉及用户数据的清理，需要提醒用户确认是否继续执行
